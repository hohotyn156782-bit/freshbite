# FreshBite — Food Delivery

Яркое приложение доставки еды. Mobile-first дизайн, интерактивное меню, система заказов.

## Стек
- HTML + CSS + Vanilla JS
- Шрифты: Nunito, DM Sans
- Тема: Тёплая, оранжево-бирюзовая

## Навыки
- Mobile-first responsive
- Интерактивная система заказов
- Категории + фильтрация
- Карусель отзывов
- CSS-анимации (floating emojis, waves)
- Двуязычность EN/RU
